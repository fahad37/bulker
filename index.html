    Features

    • User-friendly email composition interface
    • CSV upload for bulk recipients
    • Personalization with placeholders
    • SMTP configuration support
    • Works with app passwords

    How To Use

    1. Prepare a CSV with headers: email,name.
    2. Upload the CSV in Recipients.
    3. Write your subject and message. Use {{name}} to personalize.
    4. Choose EmailJS or Direct SMTP and fill required fields.
    5. Use Send Test to verify, then Send Bulk.

    Deployment

    GitHub Pages: Create a new repository, add these files, enable Pages from the repository settings with the main branch and root folder. Access the URL provided by GitHub.

    Live server: Serve this folder with any static host such as Netlify, Vercel, or a simple static server.

    About

    This tool helps send personalized emails at scale from a browser without server-side code. Data stays in your session and credentials are not stored.
